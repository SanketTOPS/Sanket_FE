Sass Examples

Sass Variables

Define reusable values for colors, fonts, and other properties.

        
        $primary-color: #3498db;
        $font-size: 16px;
        
        body {
            color: $primary-color;
            font-size: $font-size;
        }

Sass Data Types

Examples of strings, numbers, colors, booleans, lists, and nulls.

        
        $string: "Hello World";
        $number: 20px;
        $color: #ff5733;
        $boolean: true;
        $list: 1px solid red, 2px dashed blue;
        $null-value: null;

Sass Nested Rules

Allows nesting CSS selectors for better readability.

        
        .container {
            width: 100%;
            .box {
                background: #eee;
                p {
                    color: #333;
                }
            }
        }

Sass Importing Files

Use @import to split styles into multiple files.

        
        @import "variables";
        @import "buttons";
        
        body {
            font-family: Arial, sans-serif;
        }

Sass Mixins

Reusable code blocks that can accept arguments.

        
        @mixin button-style($color) {
            background: $color;
            padding: 10px 20px;
            border-radius: 5px;
        }
        
        .btn-primary {
            @include button-style(#3498db);
        }

Sass @extend Directive

Extends styles from one selector to another.

        
        .button {
            padding: 10px 15px;
            background: #ff5733;
        }
        
        .btn-danger {
            @extend .button;
            color: white;
        }
