Types of CSS Selectors

1. CSS Universal Selector

The universal selector (*) applies styles to all elements on the page.

2. CSS Element Selector

The element selector targets HTML elements like h1, p, etc.

3. CSS Id Selector

This is an example of the ID selector. IDs are unique and are applied using #.

4. CSS Class Selector

This is an example of the class selector. Classes are applied using . and can be used multiple times.

5. CSS Group Selector

This heading and the next paragraph are styled using the group selector.

Group selectors target multiple elements at once, like h2, p, to avoid repetitive CSS.

Summary

  • CSS Universal Selector: * { }
  • CSS Element Selector: element { }
  • CSS Id Selector: #id { }
  • CSS Class Selector: .class { }
  • CSS Group Selector: element1, element2 { }
