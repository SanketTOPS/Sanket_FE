Learning HTML5 Audio Tag

Introduction to HTML5 Audio Tag

The <audio> tag in HTML5 is used to embed audio content in a web page. It provides a standard way to include sound files without requiring additional plugins.

Syntax of Audio Tag

<audio controls>
    <source src="audiofile.mp3" type="audio/mpeg">
    <source src="audiofile.ogg" type="audio/ogg">
    Your browser does not support the audio element.
</audio>

Attributes used in the <audio> tag include:

  • controls: Adds play, pause, and volume controls.
  • autoplay: Automatically starts playing the audio.
  • loop: Makes the audio replay automatically after finishing.
  • muted: Starts the audio in a muted state.
  • preload: Indicates how the browser should load the audio (e.g., auto, metadata, or none).

Supported Audio Formats

The <audio> tag supports the following audio formats:

  • MP3: MPEG Audio Layer-3 (audio/mpeg)
  • WAV: Waveform Audio File Format (audio/wav)
  • OGG: Ogg Vorbis (audio/ogg)

Browser Support

The <audio> tag is supported in all modern web browsers. However, ensure to provide multiple formats for compatibility.
