Explanation of Supportive Tags

<table>
Defines the table structure.
<thead>
Groups the header content of a table.
<tbody>
Groups the body content of a table.
<tfoot>
Groups the footer content of a table.
<tr>
Defines a row in a table.
<th>
Defines a header cell in a table.
<td>
Defines a standard cell in a table.
<caption>
Provides a title or summary for the table.
<colspan> and <rowspan>
Attributes to merge cells horizontally or vertically.
