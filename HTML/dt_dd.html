HTML <dt> and <dd> Tags Example

The <dt> (definition term) and <dd> (definition description) tags are used within a <dl> (definition list) to define terms and their descriptions.

Example: Glossary

HTML
HyperText Markup Language, the standard language for creating web pages.
CSS
Cascading Style Sheets, used for styling and layout of web pages.
JavaScript
A programming language commonly used for creating interactive effects within web browsers.

Example: FAQ

What is HTML?
HTML is the standard markup language used to create web pages.
What is the purpose of the <dt> tag?
The <dt> tag is used to define a term or name in a definition list.
What is the purpose of the <dd> tag?
The <dd> tag is used to describe or define the term specified in the <dt> tag.
