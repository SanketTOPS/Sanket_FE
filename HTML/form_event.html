Form Event Attributes

onblur: Triggered when the form element loses focus.

onchange: Triggered when the value of the form element changes.

onfocus: Triggered when the form element gains focus.

oninput: Triggered when the user provides input to the form element.

oninvalid: Triggered when the form element fails to meet constraints.

onreset: Triggered when the form is reset.

onsubmit: Triggered when the form is submitted.
