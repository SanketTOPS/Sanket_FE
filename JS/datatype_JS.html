Learn JavaScript - Data Types

In JavaScript, there are different types of data types:

  • String: Used for text. Example: "Hello, World!"
  • Number: Used for numbers. Example: 42
  • Boolean: True or False values. Example: true
  • Object: Used for collections of data. Example: { name: "Sanket", age: 32 }
  • Array: A list of items. Example: [1, 2, 3]
  • Undefined: A variable that has not been assigned a value.
  • Null: Represents "nothing."
